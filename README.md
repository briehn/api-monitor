# Status Pilot

Status Pilot is a backend-focused API monitoring and uptime platform. Users will be able to add websites or API endpoints, run scheduled checks, track status and latency history, and detect incidents when services fail.

## Planned MVP
- User authentication
- Monitor creation and management
- Scheduled endpoint checks
- Stored check history with response status and latency
- Basic uptime and incident tracking
- Dashboard views for current health and recent history
